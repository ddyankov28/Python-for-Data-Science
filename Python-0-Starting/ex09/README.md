# `FT_PACKAGE`

## Build
- python setup.py sdist bdist_wheel

## Install
- pip install ./dist/ft_package-0.0.1.tar.gz
- pip install ./dist/ft_package-0.0.1-py3-none-any.whl

## Show Package
- pip show -v ft_package

## Uninstall
- pip uninstall ft_package
